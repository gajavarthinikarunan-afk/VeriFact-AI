VeriFact AI

AI-Powered Truth Verification and Misinformation Detection System

 Overview

VeriFact AI helps users verify information and identify misinformation across multiple content types including text, images and videos.

 Features

- DeepFact™ Verification Pipeline
- Truth Score Engine
- Risk Level Classification
- Real-Time Fact Checking
- Deepfake Detection
- Bias Analysis
- PDF Report Generation
- Admin Dashboard

Technologies

- Next.js 15
- React 19
- TypeScript
- PostgreSQL
- Prisma ORM
- Google Gemini API
- Tailwind CSS
- NextAuth.js
- REST APIs
- Vercel
